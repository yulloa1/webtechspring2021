Q3 - Fill out the Form (25 points)

You are give a table within a form. Read instructions carefully before starting. The first has the product name,the second is the quantity (the user will enter this) and the third, the total amount (your function needs to compute this and show it). Use Javascript to create a functionality that does the following:

1. The user should enter the quantity in one of the two rows only. If both are entered, alert the user to enter only one. If neither is entered, alert the user to enter a value in one of the two quantity fields. These two validations are worth 8 points total. For the other 5 points, validate the quantity for non-numerical values.(This is a little challenging and please make sure you get the rest working before attempting this part)

2. After entering the quantity, the user will click on the Compute Total button. Please PROMPT the user to enter a price. Make sure that you validate price for non-numerical values.You may assume that the price field will always be entered (and hence you do not have to validate it for blank value). But, the acceptable price range is between $12.99 and $39.99. If any other value is entered, please let the user know. This part is worth 8 points (for the pop up and validations).

3.Use the price and the quantity and compute the total amount and display it in the Amount field corresponding to the product that user has chosen. The amount should be shown with the $ sign. The other Amount field (i.e. for the product NOT chosen, show the amount as $0.00). Remember: You may need the "return function-name" and "return false".
